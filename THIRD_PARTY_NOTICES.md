# Third-Party Notices

## FFmpeg

`marker-fixer` may automatically download standalone FFmpeg runtime binaries (`ffmpeg` and `ffprobe`) for the current platform when local tools are not available.

FFmpeg project: [https://ffmpeg.org](https://ffmpeg.org)

License information:
- FFmpeg legal page: [https://ffmpeg.org/legal.html](https://ffmpeg.org/legal.html)
- GNU LGPL v2.1: [https://www.gnu.org/licenses/old-licenses/lgpl-2.1.html](https://www.gnu.org/licenses/old-licenses/lgpl-2.1.html)

No FFmpeg code is statically linked into `marker-fixer`; tools are used as standalone runtime binaries.
