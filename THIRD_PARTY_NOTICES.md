# Third-Party Notices

## FFmpeg

This project distributes FFmpeg runtime binaries (`ffmpeg`, `ffprobe`, and platform-specific shared libraries such as `.dll` on Windows) as standalone files.

FFmpeg is available at [https://ffmpeg.org](https://ffmpeg.org).

License information:
- FFmpeg legal page: [https://ffmpeg.org/legal.html](https://ffmpeg.org/legal.html)
- GNU LGPL v2.1: [https://www.gnu.org/licenses/old-licenses/lgpl-2.1.html](https://www.gnu.org/licenses/old-licenses/lgpl-2.1.html)

Source code for FFmpeg is available from the FFmpeg project website and official repositories linked there.

No FFmpeg code is statically linked into `marker-fixer`; FFmpeg components are distributed as separate runtime files.
